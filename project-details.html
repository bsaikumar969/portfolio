YOLO Based Adaptive Traffic Signal Control System
& Real-Time Violation Monitoring

This AI-driven system uses YOLO for real-time vehicle detection to dynamically optimize traffic signal timing and monitor violations such as red-light jumping and prioritze emergency vehicles like ambulance and firetruck. The solution improves traffic efficiency and supports smart city infrastructure. The system also provides violation evidence and GUI dashboard for controlling.

Government Schemes Eligibility Notification System

The Government Schemes Eligibility Notification System is a digital platform designed to help citizens identify and receive timely notifications about government welfare schemes for which they are eligible in there preferred language through mail. Many beneficiaries fail to access schemes due to lack of awareness, complex eligibility criteria, and scattered information across departments. This system centralizes scheme data, analyzes user profiles, and automatically matches individuals with relevant schemes based on parameters such as age, income, occupation, location, caste category, and disability status. The platform enables users to register once and receive personalized notifications whenever a new or updated scheme matches their eligibility. It improves transparency, reduces information gaps, and ensures that government benefits reach the right beneficiaries efficiently.

Tools & Technologies

  • HTML, CSS, JavaScript
  • Flask
  • PyTorch
  • MySQL
  • Email API & Translator API

Algorithms

  • Decision Tree Algorithm
  • Filtering Algorithm
  • Google Translator
  • Search Algorithm

Objectives

  • Automate Eligibility Identification
  • Ensure Timely Notifications
  • Improve Accessibility and Transparency
  • Provide Information in preffered Language
